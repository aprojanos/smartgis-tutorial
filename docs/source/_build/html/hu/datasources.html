2.3.3.1. Datasources¶

To create maps, you must first add data sources. These can be files, database connections or services from external map servers (WMS, tile servers).

2.3.3.1.1. File datasources¶

Supported vector formats

  • ESRI ShapeFile
  • ESRi Geopdatabase
  • Mapinfo TAB
  • Mapinfo MIF
  • Autocad DXF
  • GeoJSON
  • KML

Supported raster formats

  • TIFF
  • JPEG

Javaslat

Folders with raster files can be uploaded via FTP service and imported as a datasource

Megjegyzés

explain «add to application’s database», «add to project»

Javaslat

Check datasource as Basemap

2.3.3.1.3. Application database¶

The application stores spatial data in its own PostGIS database as layers. Layers can be divided into two groups: they can be edited or just used to create maps. Editing layers on the map is done with 2D/3D digitization tools. See section: Layer editor The form editor can be used to specify how the attribute table is filled. See section: Field editor

2.3.3.1.4. External databases¶

Supported databases

  • MySQL
  • PostgreSQL - PostGIS
  • MS SQL Spatial
  • Oracle Spatial
